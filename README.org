*** Q&A: 为什么不并入官方汉化？
这是一个学习之余的娱乐项目，相对于官方汉化追求严谨，本项目旨在让玩家（主要是我自己的朋友）更快的游玩到新版本（以及解决2200的地形生成bug），仅此而已，所以暂无意并入官方汉化。
